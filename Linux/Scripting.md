# Bash Script Elements

## Shebang

The shebang declares the interpreter to use for the script

```Bash
#!/bin/bash
```

## Arguments

* `$@` refers to all arguments passed in
* `$#` refers to the total number of arguments passed in

## Environmental Variables

* `PATH=<VALUE>`
