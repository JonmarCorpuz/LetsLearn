#

```Bash
crontab -e
```

```Text
<MINUTE> <HOUR> <DAY> <MONTH> <WEEKDAY> [run-parts] {<COMMAND>|<SCRIPT>|<DIRECTORY>}
```
* `run-parts` tells your server to run all the scripts within the specified directory
